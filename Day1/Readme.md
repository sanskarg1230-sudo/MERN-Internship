# Day 1 MERN Stack Internship Assignment

This repository contains my Day 1 internship assignment. The objective of this assignment was to practice modern JavaScript (ES6+) concepts and build a responsive user interface using HTML, CSS, and JavaScript.

## Tasks Completed

### Task 2.1 - JavaScript Data Processing

In this task, I worked with a mock JSON dataset containing user information. The data was parsed, validated, filtered, and transformed based on specific conditions such as age and active status.

#### Concepts Used

* JSON.parse()
* Arrow Functions
* filter()
* forEach()
* Template Literals
* Conditional Operators
* ES6 Syntax

#### Features Implemented

* Parsed JSON data
* Checked whether users are above 18 years of age
* Checked whether users are active or inactive
* Filtered eligible users (Above 18 and Active)
* Displayed processed user information in a structured format

---

### Task 2.2 - Responsive User Profile Card

In this task, I designed a modern and responsive user profile card using HTML, CSS, and JavaScript.

#### Concepts Used

* Semantic HTML
* CSS Variables
* Flexbox
* CSS Grid
* Media Queries
* Responsive Design
* JavaScript DOM Manipulation

#### Features Implemented

* Responsive layout for mobile, tablet, and desktop screens
* Light/Dark Mode Toggle
* Glassmorphism UI Design
* Hover Animations
* Adaptive Fluid Scaling
* Modern Profile Card Component
